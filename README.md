# 7.5_gotoElement.js
展示捲動位移

# 展示網址:
```
https://tzufen4115.github.io/7.5_gotoElement.js/
```

# 使用方式

1. CDN
```
<script src="https://tzufen4115.github.io/7.5_gotoElement.js/goToElement.js"></script>
```
2. 添加屬性
```
data-gte-target="目標 id"
data-gte-offset="位移 上方空間"
```
3. 目標添加ID
```
<div id="目標 ID"></div>
```
